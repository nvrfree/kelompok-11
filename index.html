Informatika

Bidang ilmu mengenai studi perancangan, dan pengembangan sistem komputasi, serta prinsip-prinsip yang menjadi dasar perancangan , pengembangan dan meningkatkan efisien dan efektif. Informatika juga disiplin ilmu yang dapat memberi pemahaman dan mengeksplorasi dunia, baik natural maupun artifisial. Mahasiswa Teknik Informatika dapat mempelajari teknologi seperti interfacing system, multimedia, Informasi Technology, Database, Applied Network.

Interfacing System



Interface adalah program yang memungkinkan pengguna untuk berinteraksi dengan perangkatnya secara langsung atau melalui jaringan. Interface juga dapat merujuk pada kontrol yang digunakan dalam program yang memungkinkan pengguna untuk berinteraksi dengan program tertentu. Interface sendiri dapat dibagi ke dalam dua kategori umum, yaitu interface dari sisi perangkat keras dan perangkat lunak.

Tools
Perangkat Lunak Pada perangkat lunak, interface merujuk kepada kontrol dalam suatu program yang memungkinkan pengguna untuk berinteraksi dengan program tersebut. Perangkat Keras Pada jenis kedua ini, contoh interface yaitu perangkat fisik, port, atau koneksi yang berinteraksi dengan komputer atau perangkat keras lain. Misalnya, IDE dan SATA adalah interface disk drive untuk hard drive komputer dan ATAPI merupakan interface untuk drive CD-ROM.

Prospek Kerja

Prospek kerja dalam bidang antarmuka (interface) sangat luas, tergantung pada spesifikasinya. Pekerjaan seperti desainer antarmuka (UI/UX), pengembang frontend, dan analis interaksi pengguna semakin dibutuhkan oleh banyak industri. Gaji bervariasi tergantung pada peran, lokasi, pengalaman, dan perusahaan, tetapi secara umum, pekerjaan ini cenderung memiliki potensi gaji yang baik. Gaji seorang desainer antarmuka (UI/UX) bervariasi berdasarkan faktor-faktor seperti lokasi, pengalaman, perusahaan, dan industri. Secara umum, di Amerika Serikat, gaji awal seorang desainer UI/UX dapat berkisar antara $50.000 hingga $70.000 per tahun. Namun, di Indonesia Gaji seorang desainer antarmuka (UI/UX) di Indonesia juga bervariasi tergantung pada faktor-faktor seperti pengalaman, lokasi, ukuran perusahaan, dan industri. Secara umum, bagi desainer UI/UX dengan beberapa tahun pengalaman, gaji bulanan dapat berkisar antara Rp 7 juta hingga Rp 15 juta atau lebih, tergantung pada faktor-faktor tersebut.

Informasi Teknologi



Informasi Teknologi (IT) adalah istilah yang mencakup segala hal yang berkaitan dengan komputer dan teknologi komunikasi dalam konteks bisnis dan pengelolaan data. Ini mencakup perangkat keras (komputer, server, perangkat jaringan), perangkat lunak (aplikasi, sistem operasi), serta infrastruktur jaringan dan layanan terkait.

Tools

Prospek Kerja

Web developer, game developer, mobile developer, AI engineer, web engineer

Tools
  • Web Developer

    Bertugas membangun suatu website atau sistem web. umumnya yg harus dikuasai web developer seperti HTML,CSS, dan JAVASCRIPT

  • Game Developer

    Secara garis besar bertugas dalam proses pemerograman dan pengembangan game digital. bahasa pemerograman yg umunya digunakan C#,C+ dan java.

  • Android Developer

    Bertanggung jawab untuk mengembangkan dan membangun perangkat lunak berbasis mobile menggunakan sistem operasi Android. Bahasa pemerograman yg biasa digunakan Java dan kotlin

Rata - Rata Gaji

  • Untuk gaji seorang web developer menurut Salary Explorer, gaji terendah web developer sebesar Rp5.000.000 per bulan dan gaji tertinggi sebesar Rp17.800.000 per bulan
  • Untuk gaji bulanan rata-rata untuk pekerjaan Game Developer di Indonesia berkisar dari Rp 6.000.000 hingga Rp 8.410.000.

Database



Database adalah koleksi data yang sistematis dan disimpan secara elektronik. Di dalam nya berisi semua jenis data, termasuk kata, angka, gambar, video, dan file. Anda dapat menggunakan perangkat lunak yang disebut sistem manajemen database (DBMS) untuk menyimpan, mengambil, dan mengedit data. Macam-macam software yang dipakai untuk database diantaranya :

  • MySql.
  • Microsoft Access.
  • Ms SQL Server.
  • Oracle.
  • Firebird.

Tools
  • Operational Database

    Jenis ini memungkinkan para pengguna dapat melakukan, melihat, dan memodifikasi data. JSON, XML

  • Database Warehouse

    sistem basis data yang biasa digunakan untuk pelaporan dan analisis data. Sistem ini dianggap sebagai komponen inti dari business intelligence.Microsoft SQL Server

  • Relational Database

    Relational Database atau basis data relasional adalah basis data yang mengorganisir berdasarkan model hubungan data. MySQl, PostgreSQL, MariaDB, MongoDB, Oracle, SAP HANA, IBM Db2, MemSQL, Interbase, Firebird

Prospek Kerja

Prospek kerja program studi teknik informatika konsentrasi bidang database yaitu,

  • Data scientist

    Data scientist adalah profesi yang menggali, menganalisis, dan menafsirkan data untuk mendapatkan insight yang dapat digunakan oleh perusahaan. Rata-rata gaji Data Scientist Menurut Payscale, rata-rata gaji seorang data scientist di Indonesia berada di rentang 7 - 380 juta rupiah per bulan, dengan rata-rata di 31 juta rupiah per bulan

  • Data Analyst

    Data analyst adalah profesi yang bertugas mengolah dan menganalisis data dari internet untuk digunakan sebagai pedoman dalam mengambil keputusan penting terkait dengan proses berjalannya bisnis. Rata-rata gaji Data Analyst Berdasarkan data dari laman job portal Indeed, gaji rata-rata untuk profesi data analyst di Indonesia adalah sebesar Rp5.083.150 per bulan.

Applied Network



Network atau jaringan adalah Sekelompok komputer yang terhubung yang bisa saling berbagi sumber daya (seperti printer atau modem) dan data. Pada defenisi sederhananya Network dapat menghubungkan setiap perangkat untuk bisa saling terhubung satu sama lainnya sehingga bisa mengirim informasi ke sasaran.

Tools

Banyak software lainnya yang dapat digunakan untuk melakukan monitoring terhadap network sehingga keamanan dari LAN atau Local Area Network lebih terjamin. Beberapa contohnya ialah layanan email seeker, Net Sender, Time Sync, Ping, Port Scanner, Port Listener, IP Calculator, IP Converter, IP Address Scanner, Online-Offline Checker, Advanced Spoofer, Advanced Port Scanner, Hack Trapper, Spoofer, Net Pager, Email Bomber, Email Spoofer, X Finger.

Prospek Kerja

Prospek karir dalam bidang networking terus menjanjikan karena terus berkembangnya teknologi dan permintaan akan konektivitas yang handal. Dengan munculnya tren seperti 5G, Internet of Things (IoT), dan komputasi awan, ahli jaringan memiliki peran yang krusial dalam merancang, mengelola, dan menjaga infrastruktur jaringan yang kompleks.

Rata - Rata Gaji

  • Pemula (0-2 tahun pengalaman): Gaji awal bisa berkisar antara 5 juta hingga 10 juta Rupiah per bulan.
  • Menengah (3-5 tahun pengalaman): Gaji rata-rata sekitar 10 juta hingga 20 juta Rupiah per bulan.
  • Berpengalaman (6+ tahun pengalaman): Gaji dapat mencapai lebih dari 20 juta Rupiah per bulan atau lebih.
